# html_rebuilder.js

html_rebuilder.js is JavaScript library to rebuild HTML by DOM.

## Usage

    <script src="js/html_rebuilder.js"></script>
    <script>
      var b = new HtmlRebuilder(document);
      prompt("Rebuiled HTML", b.html());
    </script>
